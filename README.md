# README #

### What is this repository for? ###

* COVID-19 Dashboard
* V1.0

### How do I get set up? ###

* Python -m venv venv
* venv\Script\activate (Windows)
* source venv/bin/activate (Unix)
* pip install -r requirements.txt

### How do I run? ###

* python app.py
* http://127.0.0.1:8050/
